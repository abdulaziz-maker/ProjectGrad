## ١. الحضور والغياب — أزرار جماعية وحالة ثالثة ✅

**التحسينات:**
- **حالة ثالثة "غائب بعذر"** (أصفر) بدلاً من "متأخر"؛ الحالات الآن ثلاث فقط:
  حاضر (أخضر) · غائب (أحمر) · غائب بعذر (أصفر).
- **زر "تحضير الكل"** يحدد جميع الطلاب حاضرين بنقرة.
- **زر "تغييب الكل"** للإجازات.
- **زر "غياب بعذر للكل"** لليوم الذي تجمع فيه الظروف كل الطلاب بعذر.
- البيانات القديمة المحفوظة بقيمة `late` تُعرض تلقائياً كـ "غائب بعذر"
  وتُحوَّل عند إعادة الحفظ. مايجريشن SQL يرحّل السجلات القديمة وينشئ
  CHECK constraint يمنع أي حالة غير الثلاث المذكورة.
- **تصحيح نسبة الحضور**: السابق كان يعدّ `late` ضمن الحضور، والآن الغياب
  بعذر يُستبعد من الحسبة كلياً (لا يُعد حضوراً ولا غياباً) ليعكس الواقع.

**الملفات:**
- `app/(main)/attendance/page.tsx` (UI + حساب)
- `lib/types.ts`, `lib/store.ts` (توسيع union type)
- `supabase-attendance-excused-migration.sql` (جديد — يجب تشغيله)

## ٥. رصد المتون — المستوى التمهيدي + أبواب + تعديل المعدل ✅

**أ. المستوى التمهيدي (level 0):**
- `LEVEL_LABELS[0] = 'التمهيدي'` في `matn/page.tsx` و `matn/manage/page.tsx`.
- `LEVEL_IDS = [0,1,2,3,4,5,6]` بديلاً عن المصفوفة المضمنة.
- سياسة CHECK في DB موسَّعة: `level_id BETWEEN 0 AND 6`.
- بطاقات المستويات الآن ٧ أعمدة (`grid-cols-7`).

**ب. تعديل مقدار التسميع:**
- `weekly_rate` كان قابلاً للتعديل في نموذج إضافة/تعديل المتن، لكن لم يُعلَن
  عنه صراحة في الوثائق. وثّقناه هنا. الحقل `FormField` في `matn/manage/page.tsx`
  يقبل رقماً (≥ ١) ويُستخدم في `generate_text_units` لتوليد المقررات.

**ج. دعم الأبواب (text_chapters):**
- جدول جديد بالأعمدة: `id, text_id, chapter_number, title, start_line, end_line`
  مع `UNIQUE (text_id, chapter_number)` و `CHECK (end_line >= start_line)`.
- RLS: قراءة لكل مسجّل، كتابة لـ `ceo` فقط.
- دوال DB: `getTextChapters`, `upsertTextChapter`, `deleteTextChapter` في
  `lib/db.ts`.
- (ملاحظة: لم تُنفَّذ واجهة الأبواب في هذه الدفعة — الأساس جاهز.)

**الملفات:**

## اختبار يدوي مقترح

| السيناريو | النتيجة المتوقعة |
|-----------|------------------|
| دخول الحضور → اضغط "تحضير الكل" | كل الطلاب يصبحون حاضرين بلون أخضر |
| دخول الحضور → اضغط "غائب بعذر" على طالب | اللون أصفر، العدّاد يتحدّث |
| `manager/supervisors` كمدير دفعة → حضور المشرفين يظهر | أسفل العنوان بطاقة حضور |
| `manager/reports` → بطاقة "متابعة الأسبوع" | تظهر بالنسبة الصحيحة |
| `manager/reports` → زر "لم تتم متابعتهم فقط" | يفلتر الطلاب المتابعين |
| `batches` → أيقونة CheckCircle2 فوق جزء → تأكيد → حفظ | كل الطلاب في الجزء يصبحون "محفوظ" |
| `matn` → تبويب "التمهيدي" | يظهر كأول تبويب |
| `exams` → "الأسبوع السابق" | التقويم يعود ٧ أيام |
| `exams` كمشرف دفعة ٤٨ → اختبار دفعة ٤٦ | يظهر رمادي مع "قراءة فقط" |
| `followups/plan/[studentId]` → عدّل "المراجعة القريبة" → حفظ | يظهر toast النجاح |
